# Max - AI Voice Assistant

A modern Android voice assistant app with Gemini API integration.

## Features

- Voice Commands (Calls, SMS, Weather, Music, Search)
- AI Chatbot with Gemini API
- Custom API Key Support
- Multi-language Support (English, Hindi, Hinglish)
- Modern Dark UI Design

## Setup

1. Get Gemini API Key from [Google AI Studio](https://makersuite.google.com)
2. Add API key in app Settings
3. Start chatting with Max!

## Build with Codemagic

1. Connect GitHub repo to [Codemagic](https://codemagic.io)
2. Select Flutter workflow
3. Build APK automatically
